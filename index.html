Sobre Mí

Soy un Desarrollador Java Full Stack Jr. con una sólida formación en desarrollo de software y una pasión por crear aplicaciones web eficientes y bien estructuradas.

Mi enfoque principal está en el desarrollo backend con Java y Spring Boot, donde diseño y construyo APIs REST robustas conectadas a bases de datos MySQL. También tengo experiencia en el frontend utilizando HTML, CSS, JavaScript y Bootstrap para crear interfaces de usuario responsivas y atractivas.

Me apasiona aprender nuevas tecnologías y mantenerme actualizado con las mejores prácticas de desarrollo. Utilizo herramientas como Git, GitHub, Postman y Maven para mantener un flujo de trabajo profesional y organizado.
